Colour can be changed using the following:

  • Colour word e.g. 'red'
  • Colour HEX code e.g. '#dfd3c3'
  • MDN's list of detailed colour names e.g. 'coral'

Check the Atom body tag for code needed to change the colour.

Page colour can be set in two different ways. See examples in the code of the head and body.
